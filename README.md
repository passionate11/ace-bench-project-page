# Agentic Edge-Cloud Collaboration Benchmark

Static project page and leaderboard for ACE-BENCH.

## Local Preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Data Updates

Leaderboard entries live in `data/results.json`. The page sorts by Completion Score by default while keeping cloud cost and privacy visible in both the table and the trade-off plot.

## Deployment

This repository is designed for GitHub Pages. Once the private repository is ready, enable Pages from the repository settings and select the default branch as the source.
