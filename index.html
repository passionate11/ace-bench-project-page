Evaluation Profile

ACE-BENCH reports every strategy as a utility, cost, and privacy profile.

The benchmark avoids hiding deployment trade-offs inside a single scalar score.

Utility

Can the agent finish the task?

Completion Score and Pass@3 evaluate whether multi-step workspace goals are actually solved.

Resource Usage

How much cloud and edge compute is used?

Cloud tokens, estimated API cost, and edge-side FLOPs expose the execution burden.

Privacy Performance

What local context reaches the cloud?

Risk-weighted non-leakage tracks sensitive units visible to cloud models along the trajectory.

Key Findings

Cloud-only systems deliver strong task utility, but expose much more local context. Edge-only systems preserve privacy, yet lose capability on long-horizon workspace tasks.

Edge-cloud collaboration improves the operating point by selectively invoking cloud capability, but the result depends on when the cloud is used and what context is transmitted.

Stronger edge models shift the preferred pattern from step-level fallback toward task routing, planning, and recovery guidance.
